# Traceroute Visualization System
This project is a GUI-based Traceroute Visualization System developed to analyze and understand the network path between a source and a destination. The application performs a traceroute operation for a given domain name or IP address and displays all the intermediate nodes (hops) along with their response time, helping users visualize how data travels across a network. Built using Python and basic networking concepts, the system provides a simple and user-friendly interface for real-time network path analysis. It is especially useful for students and beginners to understand routing behavior, delays, and the working of traceroute in a clear and practical way. To run the project, ensure Python is installed, open the project folder, and execute the main file using the command `python Mainfile.py`, after which the user can input a domain or IP address to view the traceroute results.
